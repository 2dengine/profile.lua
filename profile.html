profile

Simple profiler written in Lua.

profile.hooker(event, line, info)This is an internal function.
profile.query(limit)Iterates all functions that have been called since the profile was started.
profile.report(limit)Generates a text report.
profile.reset()Resets all collected data.
profile.setclock(func)Sets a clock function to be used by the profiler.
profile.start()Starts collecting data.
profile.stop()Stops collecting data.
